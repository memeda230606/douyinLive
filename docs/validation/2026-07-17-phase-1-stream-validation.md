## 4. 结论

当前 web/enter 链路可以从真实在线房间稳定取得 FLV/HLS 候选，并由 FFmpeg 直接读取 H.264/AAC 媒体。候选 ID 在签名地址刷新时保持稳定，适合写入媒体清单；完整 URL 必须只作为短生命周期内存快照使用。

本次在约 23 秒窗口内未观察到旧地址失效，但观察到刷新请求返回了不同的完整地址。因此录制器不得长期缓存 URL：输入出现 401、403、404、410、立即 EOF 或连续连接失败时，应重新调用 ResolveStreams，按排序后的候选序列降级，而不是无限重试旧地址。
